# LearnWords
An Android application for learning foreign words.

# Build

## Debug

1. Install Android Studio
2. Install SDK 27
3. Clone the project
4. Open it in AndroidStudio
5. Click 'Sync Now' if gradle suggested it
4. Connect your phone
5. Press Shift+F10
6. Select your phone
7. ?????
8. Profit!
